## Бот для управления поэтическими вечерами

Телеграм-бот для автоматизации организации поэтических мероприятий. Это мой первый проект, который я разработала с нуля за 2 недели.
Этот бот помогает организовать поэтические вечера. Участники могут подавать заявки со своими стихами, а организаторы — управлять всем процессом в одном месте.

## Что умеет бот

**Для участников:**
- Подавать заявки со своими стихотворениями
- Выбирать участие в первом или втором блоке вечера
- Читать правила и информацию о мероприятии
- Получать уведомления о статусе своей заявки

**Для организаторов:**
- Просматривать и проверять все заявки
- Утверждать или отклонять стихотворения
- Экспортировать списки участников и тексты стихов
- Делать рассылку всем участникам
- Управлять черным списком
- Редактировать правила и информацию о мероприятии прямо в боте

## Как это работает технически

- Написан на Python с использованием библиотеки python-telegram-bot
- Данные хранятся в SQLite базе
- Конфигурация через файл .env
- Логирование всех действий

## Как запустить у себя

1. **Установите Python 3.8 или новее**

2. **Установите нужные библиотеки:**
```bash
pip install python-telegram-bot python-dotenv
```

3. **Настройте бота:**
   - Создайте бота в Telegram через @BotFather
   - Узнайте свой ID в Telegram (можно через @userinfobot)
   - Создайте файл .env и добавьте туда:
     
```BOT_TOKEN=токен_вашего_бота

ADMIN_ID=ваш_telegram_id```

4. **Запустите бота:**

bash
python main.py


## Как пользоваться

**Если вы участник:**
1. Напишите боту /start
2. Нажмите "Подать заявку"
3. Отправьте текст своего стихотворения
4. Выберите, хотите ли выступать во втором блоке
5. Готово! Ваша заявка отправлена на проверку

**Если вы организатор:**
1. Напишите /start
2. Нажмите "Меню Организатора"
3. Вам откроется панель управления где вы можете:
   - Проверять новые заявки
   - Экспортировать списки участников
   - Делать рассылки
   - Управлять черным списком
